# Line5-Android

右侧（或在最下方）Releases处下载apk ->

## 游戏简介

致敬经典的DOS游戏Color Lines。

- 经典同色消除游戏，有7种不同颜色的球。
- 满足横向、纵向、斜向至少5个同色球即可消除得分。
- 若玩家有自信，可关闭下一轮颜色显示。
- 易于上手，难于精通。
